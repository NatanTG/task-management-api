## Description

This repository demonstrates a **CRUD for task management** developed with **NestJS**. The project leverages **Docker**, **Prisma ORM**, and **PostgreSQL** to build a robust and scalable backend. 

It includes a complete implementation of Create, Read, Update, and Delete operations while adhering to clean architecture principles.

---

## Features

- **Task Management**: Create, list, update, and delete tasks.
- **Prisma ORM**: Simplified database interactions.
- **PostgreSQL**: Relational database for efficient data handling.
- **Docker Support**: Quick and consistent environment setup.
- Clean architecture with separation of concerns.
